Hello PIO
=========

Simple "Hello world" example for the RP2040 microcontroller, using PIO.

See:

* https://www.raspberrypi.com/news/what-is-pio/
* https://datasheets.raspberrypi.com/pico/raspberry-pi-pico-c-sdk.pdf
* https://gregchadwick.co.uk/blog/playing-with-the-pico-pt4/

For more information on building, see the [hello-world](../hello-world)
directory.
